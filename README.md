# 🚲 E-shop s koly - seminární projekt

**Autoři:** Serhii Chornobai, Maksym Polyvoda, Jakub Tuček

---

## 📌 O projektu

Databázový systém pro e-shop s jízdními koly a správu servisních zakázek. Systém pokrývá:

- Katalog kol s kategoriemi
- Nákupní košík a objednávky
- Servisní zakázky s úkony
- Zaměstnance a reporting

## 📎 Soubory:

- [`create.sql`](./create.sql) - definice tabulek (DDL) - vytvoření databázového schématu
- [`insert.sql`](./insert.sql) - testovací data (DML) - naplnění tabulek vzorovými daty
- [`queries.sql`](./queries.sql) - ukázkové SQL dotazy pro různé scénáře
